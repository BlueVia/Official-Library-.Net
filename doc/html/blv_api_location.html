LOCATION API GUIDE

LOCATION:

The Bluevia Location API provides access to the geographical coordinates of a terminal where Location is expressed through a latitude, longitude, and accuracy. The operations cover the requesting for the location of a terminal or a group of terminals.
This guide represents a practical introduction to developing applications in which the user wants to provide the Bluevia Location functionality.

If you want to understand how Location API works please read the BlueVia Bluevia's Location API guide.

Don't forget to read the authorization guide first.

This guide is divided into the following sections:

  • Location Client.
  • Location Methods.
    • Retrieving Geographical coordinates of a terminal.

 

Location Client

A Location client represents the client side in a classic client-server schema. This object wraps up the underlying REST client side functionality needed to perform requests against a REST server.

Creating a Location client:
Bluevia.Location.Client.BV_Location.BV_Location

  • Location Client: 
            BV_Location locationClient = new BV_Location(BVMode.MODE, consumerKey, consumerSecret, token, tokenSecret);

 

Location Methods:

Retrieving Geographical coordinates of a terminal:

The getLocation function allows an user to retrieve the geographical coordinates of a terminal. These geographical coordinates are expressed through a latitude, longitude, altitude and accuracy.

If you want to understand how Retrieving the Geographical coordinates works, please read the BlueVia Location API guide.

  • The accuracy parameter expresses the range in meters that the application considers useful. If the location cannot be determined within this range, then the application would prefer not to receive the information.

Retrieving Geographical coordinates of a terminal:
Bluevia.Location.Client.BV_Location.GetLocation 

        try
        {
                LocationInfo locationInfo = locationClient.GetLocation
                (
                        accuracy:500 //Optional
                );
        }
        catch (BlueviaException e){}

Processing the response:
Bluevia.Location.Schemas.LocationInfo
Once the server responds the user have to retrieve the location information from the returned LocationInfo instance. The LocationInfo includes the status of the client request and the the location information.
