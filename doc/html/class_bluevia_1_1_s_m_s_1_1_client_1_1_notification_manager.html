 Start notifications to the application for a given SMS Destination Address and criteria. The correlator provided in the reference must be unique for the application Web Service at the time the notification is initiated. If specified, criteria will be used to filter messages that are to be delivered to an application. If criteria are not provided, or is an empty string, then all messages for the Destination Address will be delivered to the application.Note that the use of criteria will allow different notification endpoints to receive notifications for the same Destination Address. The combination of Destination Address and criteria must be unique, so that a notification will be delivered to only one notification endpoint.
