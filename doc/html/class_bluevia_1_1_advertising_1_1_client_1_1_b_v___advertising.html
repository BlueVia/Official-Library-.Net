Requests the retrieving of an advertisement. This function can only be used in 2-legged-mode (To use when the OAuth token hasn't been included in the construction of the client).

Parameters:
adSpaceThe adSpace of the Bluevia application.
countryCountry where the target user is located. Must follow ISO-3166.
targetUserIdOptional: Identifier of the Target User.
adRequestIdOptional: an unique id for the request. (if it is not set, the SDK will generate it automatically).
adPresentationOptional: The value is a code that represents the ad format type
keywordsOptional: If you wish to retrieve advertisings related to some keywords.
protectionPolicyOptional: The adult control policy. It will be safe, low, high. It should be checked with the application SLA in the Bluevia endpoint
userAgentOptional: The user agent of the client. ("none" by default).
Returns:
The result returned by the server that contains the advertising meta-data
