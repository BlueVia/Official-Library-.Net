Detailed Description

The BV_HTTP_OAuth_Connector: Class wich extends the HttpConnector functionallity, providing OAuth authentication info to access the BlueVia services in Http way. It receives the invocation from the Bluevia client, formats the received data, and executes the HttpConnector's relative method.
