Schemas.SMSDeliveryStatusType Bluevia::SMS::Client::MessageMT::GetStatus ( string  messageId) [inline]

The application invokes the GetStatus operation to request the status of a previous SMS delivery request identified by messageId. The information on the status is returned in SmsDeliveryStatusType, which is an array of status related to the request identified by MessageId. The status is identified by a couple indicating a user address and the associated delivery status. This method can be invoked multiple times by the application even if the status has reached a final value. However, after the status has reached a final value, status information will be available only for a limited period of time as defined by a service policy.
