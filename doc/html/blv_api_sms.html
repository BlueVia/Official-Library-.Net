SMS API

SMS:

Bluevia's SMS API Guide

The Bluevia SMS API is a set of functions which allows users to send and receive SMS messages and request the status of those previously sent messages. Also you can subscribe notifications to your own server.

Note that SMS_MO requests are now made by 2leggedOauth mode (not authorized clients)

Objects:

  • UserIdTypes:

    Consider that for this api, the UserIdType objects must match the phoneNumber category: 

            UserIdType destination = new UserIdType { 
        Item = "the destination number", ItemElementName = Bluevia.SMS.Schemas.ItemChoiceType1.phoneNumber };
  • Creating a message object: 
            Bluevia.SMS.Schemas.SMSTextType message = new Bluevia.MMS.Schemas.MessageType
        (new string[] {"Destination 1", "Destination 2"...},"Message text"*optional);
  • Creating a MessageNotification object: 
            Bluevia.SMS.Schemas.SMSNotificationType notification = new SMS.Schemas.SMSNotificationType()
        {
        reference = new SimpleReferenceType() { correlator = "A correlator", endpoint = "The endpoint" },
    destinationAddress = new UserIdType[] { new UserIdType() { Item = "The short Code", ItemElementName = ItemChoiceType1.phoneNumber } },
    criteria = "The criteria"
        };

Methods

  • Sending SMS:
    • First mode: 
                      var response = authenticatedRequest
                .SMS.MessageMT.Send(new string[] {"Destination 1","Destination 2"...    },"Message text");
    • Second mode: 
                      var response = authenticatedRequest
                .SMS.MessageMT.Send(message);
  • Getting Status: 
            var response = authenticatedRequest
        .SMS.MessageMT.GetStatus("messageId", "status"*optional);
  • Getting SMS: 
            var response = request
        .SMS.MessageMO.GetMessages("registrarionId");
  • Subscribe Notifications: 
            var response = request
        .SMS.NotificationManager.Subscribe(notification);

  • Unsubscribe Notifications: 

            var response = request
        .SMS.NotificationManager.UnSubscribeNotification("notificationId");
